# Requirements Definition

## Problem

初心者の時にエラーを読むことができない。

今の時代AIがあるからAIが直してくれるけどこれでは勉強にならない。

エラーを読めなくても理解して自分でコードを直して成長したい。

## Solution

エラーメッセージを入力するとどういうエラーなのかを解説してもらえる。

わからなかったら、コード例を交えていろんな場面ごとにどうすればいいか説明をしっかりしてくれるアプリ

プログラミング学習補助アプリ

# Overview

| 項目 | 内容 |
| --- | --- |
| **プロダクト名** | wat!? |
| **一言説明** | エラーメッセージを貼るだけで、何が起きてるか日本語で教えてくれる |
| **ターゲット** | プログラミング初心者（歴1年未満） |
| **解決する課題** | エラーメッセージが読めない・意味が分からない |

## User Story

```
プログラミング初心者として、
エラーメッセージを入力したら、
何が起きているか・どう直すかを日本語で知りたい。
なぜなら、英語のエラーメッセージが読めないから。
```
